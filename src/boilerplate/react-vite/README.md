# React (Vite) Starter

Vite + React 19 + TanStack Router + Tailwind v4 + shadcn/ui + react-i18next + sonner.

## Local development

```bash
pnpm install
pnpm dev
```

| Script | Description |
|--------|-------------|
| `pnpm dev` | Dev server with HMR (also generates `src/routeTree.gen.ts`) |
| `pnpm build` | Typecheck + production build → `dist/` |
| `pnpm preview` | Preview the production build |
| `pnpm lint` | ESLint |

> `src/routeTree.gen.ts` is generated by the TanStack Router Vite plugin the first time you run `pnpm dev` or `pnpm build`. It is not checked into this template — running the dev server creates it automatically.
